Project 1: Loan Portfolio Performance Dashboard
Overview

This project analyses a loan portfolio to evaluate overall lending performance, loan distribution, and risk exposure. The dashboard provides key insights into total loan value, loan volume, average loan size, and loan status performance to support data-driven decision-making.

Key Business Questions

What is the total value of loans issued?

How many loans are currently in the portfolio?

What is the average loan amount?

How are loans distributed by loan status (approved vs not approved)?

Key Metrics (KPIs)

Total Loan Amount

Number of Loans

Average Loan Amount

Visualisations

KPI cards for total loan amount, loan count, and average loan size

Bar chart showing loan amount by loan status

Bar chart showing number of loans by loan status

Interactive slicer for loan status filtering

Tools & Techniques

Power BI Desktop

DAX measures (SUM, COUNT, AVERAGE)

Data modelling and aggregation

Interactive dashboards and filters

Outcome

The dashboard enables quick assessment of loan exposure and approval patterns, helping stakeholders understand portfolio health and lending trends at a glance.
